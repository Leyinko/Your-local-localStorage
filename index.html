INTRODUCTION

Local Storage is a web browser feature that allows web applications to store data on the user's device.

It's a key-value storage system that persists even after the browser is closed.


This app lets you store data into Local Storage and should automatically recognize Strings, Numbers and Booleans.

You can choose between three different storing data types :


ITEMS, ARRAYS, AND OBJECTS.


The virtual box will display the current Local Storage content.

You can properly access it via the Developer Tools in your browser.




    GET STARTED


  1. 1. CHOOSE A DATA TYPE.
  2. 2. INPUT DATA WITH THE CORRECT SYNTAX (NO NEED FOR SYMBOLS).
  3. 3. CLICK "ADD ITEM" OR PRESS ENTER TO STORE IN LOCAL STORAGE.
