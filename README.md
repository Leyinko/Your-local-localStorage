<p style="font-weight: bold">Overview :</p>
<ol>

- The input should recognize Strings, Booleans, and Numbers, and based on the syntax used, it stores data into single key/value Items, Arrays, or Objects; no need for symbols!
- Completely useless, but could help explore and learn a bit about the Local Storage API.
- Personal design (Use of PS).
- Tablet and mobile versions.
- Originally intended as a test for the "Storage" Event Listener, the exploration of Local Storage unexpectedly transformed it into the second project >
